# ML Server — TravelGenius Cost Predictor

FastAPI-based machine learning server for trip cost prediction.

## Setup

```bash
pip install -r requirements.txt
python train_model.py    # Generate dataset & train model
uvicorn app:app --port 8000 --reload
```

## Endpoints

| Method | Path | Description |
|--------|------|-------------|
| POST | `/predict` | Predict trip cost |
| GET | `/health` | Health check |
| GET | `/model-info` | Model metadata |

## API Docs

Once running, visit: http://localhost:8000/docs (auto-generated by FastAPI)
